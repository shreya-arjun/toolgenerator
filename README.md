# ToolGenerator

Offline multi-agent tool-use conversation generator.

Internship assessment project.

## Installation

1. `git clone <repo>`
2. `cd toolgenerator`
3. Download ToolBench data from Google Drive and place at `data/toolenv/tools/`
4. `pip install -e .`
5. Run: `toolgenerator build --tools-path data/toolenv/tools`
